# Gap Buffer Editor

This project implements a command-line text editor using
the Gap Buffer (Two Stack) approach.

## How to Run
* Make sure python 3 is installed
* Open terminal / command line
* Go to the folder containing gap_buffer_editor.py
* Run `python src\gap_buffer_editor.py`
* Type commands:
      -Type → insert character
      -LEFT / RIGHT → move cursor
      -SHOW → display current text
      -BACKSPACE / DELETE → remove characters
      -EXIT → quit the program 

## Data Structures
- Two stacks (Left and Right of cursor)
